# AuraTech

AuraTech is a cyberpunk-inspired mobile productivity and journaling application built through AI-assisted development.

## Features

* XP-based leveling system
* Daily habit tracking
* Quest-based task management
* AI-inspired motivational guidance
* Journal and mood tracking
* Cyberpunk user interface
* Local data persistence
* Story/Reel export functionality

## Tech Stack

* React Native
* Expo
* TypeScript

## Why I Built This

This project was built to explore how AI-assisted development can accelerate the journey from idea to working mobile application.

The focus was not only on generating code, but also on product design, prompt engineering, validation, testing, and iterative refinement.
